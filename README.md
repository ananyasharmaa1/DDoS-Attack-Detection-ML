# DDoS Attack Detection using Machine Learning

This project focuses on detecting Distributed Denial of Service (DDoS) attacks using machine learning algorithms.

## Algorithms Used
- Random Forest
- LSTM
- Neural Network
- Random Classifier (baseline)

## Tools & Technologies
- Python
- Google Colab
- Scikit-learn
- TensorFlow
- Matplotlib

## Project Workflow
1. Data preprocessing and feature selection
2. Training multiple machine learning models
3. Evaluating models using accuracy metrics
4. Comparing model performance through visualizations

## Result
The machine learning models significantly outperform the random classifier, showing strong capability in detecting DDoS attack patterns.
